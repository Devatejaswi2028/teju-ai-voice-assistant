# Teju AI Voice Assistant

Teju AI Voice Assistant is a Python-based intelligent desktop assistant designed to interact with users through voice commands. The assistant can recognize speech, respond with natural voice output, provide real-time weather updates, tell the current time and date, and perform basic smart assistant tasks.
